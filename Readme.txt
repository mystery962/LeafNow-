
###Leaf Now

- Responsive Plants Website Using HTML, CSS and JavaScript.
- Contains animations when scrolling.
- Includes a dark and light mode. 
- Compatible with all mobile devices and with a beautiful and pleasant user interface.

Instructions to run for a new user:

-Extract the file and open the folder using Visual Studio Code
-Have django and python modules installed.
-Enter python manage.py runserver in the terminal.
-You will be prompted with the login page , if you are new to the website sign up by clicking the 'New User? Sign Up' option. Then enter your personal details that 
 the website requires you to fill. Remember your username and password as it is necessary for logging in.
-After you have completed signing in, you can enter the correct username and password which will redirect you to the main page(index.html) of the website.
